Safety Box Usage Instructions
Close: choose and enter personal code followed by (#)
Open: enter personal code followed by (#)
Step 1: To lock the safety box, choose a 4-6 digit personal code, enter it, then press the # button.
Step 2: To unlock the safety box, enter the same personal code, then press the # button.
Step 3: Ensure the door is properly closed after locking.
Step 4: If the wrong code is entered multiple times, the safety box will temporarily lock for added security.
Important Notice:
- Please do not leave the safety box door open when not in use.
- Memorize your personal code or store it securely.
- In case of any issues, contact hotel reception for assistance.
- The hotel is not responsible for any loss of valuables left outside the safety box.
